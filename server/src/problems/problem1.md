# A + B

##### 시간 제한: 1초, 메모리 제한: 512MB

## 문제

두 정수 $A$, $B$가 주어질 때, $A + B$를 출력하는 프로그램을 작성하시오.

## 입력

첫 번째 줄에 두 정수 $A$, $B$ $(|A|, |B| \leq 10^{18})$가 공백을 사이에 두고 주어진다.

## 출력

첫번째 줄에 답을 출력한다.

## 예제 입력 1

```
1 1
```

## 예제 출력 1

```
2
```

## 예제 입력 2

```
12 -12
```

## 예제 출력 2

```
0
```

## 예제 입력 3

```
123 218
```

## 예제 출력 3

```
341
```
